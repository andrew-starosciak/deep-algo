## Instructions

Produce a comprehensive battle plan for the coming week:

### 1. Macro View
- Current regime: risk-on or risk-off?
- Key economic events this week (CPI, FOMC, jobs, etc.)
- How should we position for the macro environment?

### 2. Sector Analysis
- Which sectors showed relative strength/weakness this week?
- Any sector rotation signals?
- Which sectors have the best risk/reward for next week?

### 3. Performance Review
- How did our trades perform this week?
- What worked and what didn't?
- What patterns can we identify?
- Concrete lessons to apply going forward

### 4. Top Ideas for Next Week
- List 3-5 specific ticker ideas with brief thesis
- Rank by conviction
- Note any that are time-sensitive (earnings week, etc.)

### 5. Focus Tickers
- Which tickers should we monitor most closely?
- What price levels or events would trigger action?

Be specific and actionable. This plan drives the entire coming week.
